Programming Languages



C++

C++ is a cross-platform language that can be used to create high-performance applications. C++ was developed by Bjarne Stroustrup, as an extension to the C language.

Java

Java is a case-sensitive programming language, like C++. Java is an Object-Oriented Programming (OOP) structure. Java is a class-based programming language.

Python

Python has a simple syntax similar to the English language. Python has syntax that allows developers to write programs with fewer lines than some other programming languages.
